CloudEnv

public class CloudEnv

CloudEnv class

Convenience class for obtaining environment variables that are mapped to JSON strings. Mainly used for obtaining credentials for services so Swift applications can be written in a platform agnostic way.

    public init(mappingsFilePath: String? = nil, cloudFoundryFile: String? = nil)

    Parameters

    mappingsFilePath

    Optional. The path to the mappings.json file; this path should be relative to the root folder of the Swift application.
    cloudFoundryFile

    Optional. The path to a JSON file that contains values for Cloud Foundry environment variables (mainly used for testing); this path should also be relative to the root folder of the Swift application.
